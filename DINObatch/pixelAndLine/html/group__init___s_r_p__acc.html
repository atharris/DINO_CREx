A script used to run the acc batch without SRP in the reference EOMs.

Overview

Purpose

This module is nearly identical to initAcc - in house batch initialization script for unmodeled accleration estimation. As in init_SRP_test - an initialization script for compare_SRP_test.py , the sole difference from is an implemented change to remove SRP from the EOMs used in the creation of a reference trajectory. Because of this, the script is one piece of a methodology for tracking the effectiveness of the batch filter software in the face of incomplete reference EOMs. This script is called so that compare_SRP_test - a test of the filter with and without SRP can unpack the results and compare them to other approaches.

The Code

main

In order to set the stage for the comparison test of compare_SRP_test - a test of the filter with and without SRP , this function runs a batch filter with SRP while creating measured data, but leaves out this force model in the reference propagator. Because of this ommission, the results can be compared with outputs of init - in house batch initialization script , initAcc - in house batch initialization script for unmodeled accleration estimation , and init_acc_SRP_test - an initialization script for compare_SRP_test.py . This comparison can then be used to evaluate the performance of all four approaches.

The first and most signficiant difference between this script and that of initAcc - in house batch initialization script for unmodeled accleration estimation is the line:

# alter to coefficient of reflectivity to be zero. This negates any contribution of
# modeling SRP
extras['cR'] = 0.0

This line, positioned just before the filter is run, effectively removes any SRP model from the reference propagator. Because the measured data has already be created, the observations have been propated with SRP. The only other difference is the file tag used in pickling:

# Write the output to the pickle file
fileTag = 'SRP_test'
file = dirIt+'/'+fileTag+'_data.pkl'

This line simply saves the results of this script to files with names different than those of initAcc - in house batch initialization script for unmodeled accleration estimation ("nominal").
